===============
API - Endpoints
===============

This section describes the available endpoints of the API

+-------------+----------------------+-------------------------------------------+
| HTTP method | URI                  | Description                               |
+=============+======================+===========================================+
| **GET**     | /instruments         | Returns the list of Instruments           |
+-------------+----------------------+-------------------------------------------+
| **GET**     | /instruments/2       | Returns the Instrument referenced by *2*  |
+-------------+----------------------+-------------------------------------------+
