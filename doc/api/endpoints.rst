===============
API - Endpoints
===============

This section describes the available endpoints of the API

.. code::

**/instruments**

+-------------+------------+-------------------------------------------+
| HTTP method | URI        | Description                               |
+=============+============+===========================================+
| **GET**     | /          | Returns the list of Instruments           |
+-------------+------------+-------------------------------------------+
| **GET**     | /2         | returns the Instrument referenced by *2*  |
+-------------+------------+-------------------------------------------+
